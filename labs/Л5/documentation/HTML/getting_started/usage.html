Использование TaskFinance

Обзор

TaskFinance — это мощный планировщик задач с встроенной системой финансового учета. С помощью этого приложения вы можете управлять задачами, отслеживать финансовые операции и просматривать историю действий.

Основные функции

  • Создание, редактирование и удаление задач.
  • Финансовые расчёты, связанные с задачами.
  • История изменений с фильтрацией по временным периодам.

Работа с задачами

Добавление задачи

Чтобы добавить новую задачу, выполните следующие шаги:

  1. Запустите приложение.
  2. Выберите опцию для добавления задачи.
  3. Введите название и описание задачи.
  4. Задача будет добавлена в список.

Обновление статуса задачи

Чтобы обновить статус задачи:

  1. Выберите задачу из списка.
  2. Введите новый статус (например, "выполнено").
  3. Статус задачи будет обновлён.

Работа с финансами

Добавление финансовой записи

Чтобы добавить финансовую запись, выполните следующие шаги:

  1. Запустите приложение.
  2. Выберите опцию для добавления финансовой записи.
  3. Введите ID задачи, сумму и категорию (Доход/Расход).
  4. Запись будет добавлена в систему.

Просмотр финансового отчета

Чтобы просмотреть финансовый отчет:

  1. Запустите приложение.
  2. Выберите опцию для отображения финансового отчета.
  3. Вы увидите общую сумму доходов, расходов и текущий баланс.

История действий

Вы можете просмотреть историю действий, чтобы отслеживать изменения, связанные с задачами и финансами. Для этого выберите соответствующую опцию в меню приложения.
