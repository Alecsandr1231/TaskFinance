Документация TaskManager

Обзор

Класс TaskManager отвечает за управление задачами в приложении TaskFinance. Он предоставляет методы для создания, редактирования, удаления и получения задач.

Методы

create_task(title, description)

Создает новую задачу.

  • Параметры:
    • title: Заголовок задачи.
    • description: Описание задачи.
  • Возвращает: Идентификатор созданной задачи.

get_all_tasks()

Возвращает список всех задач.

  • Возвращает: Список всех задач.

update_task(task_id, new_status)

Обновляет статус существующей задачи.

  • Параметры:
    • task_id: Идентификатор задачи, статус которой нужно обновить.
    • new_status: Новый статус задачи.

delete_task(task_id)

Удаляет задачу по указанному идентификатору.

  • Параметры:
    • task_id: Идентификатор задачи, которую нужно удалить.

Пример использования


# Пример создания новой задачи
task_id = TaskManager.create_task(title='Новая задача', description='Описание новой задачи')

# Получение всех задач
tasks = TaskManager.get_all_tasks()

# Обновление статуса задачи
TaskManager.update_task(task_id=task_id, new_status='выполнено')

# Удаление задачи
TaskManager.delete_task(task_id=task_id)
