Документация FinanceController

Обзор

Класс FinanceController отвечает за управление финансовыми записями в приложении TaskFinance. Он предоставляет методы для добавления записей, получения всех записей, вычисления сумм по категориям и получения текущего баланса.

Методы

add_record(task_id, amount, category)

Добавляет новую финансовую запись, связанную с задачей.

  • Параметры:
    • task_id: Идентификатор задачи, с которой связана запись.
    • amount: Сумма финансовой операции.
    • category: Категория финансовой операции (например, "Доход" или "Расход").

get_all_records()

Возвращает список всех финансовых записей.

  • Возвращает: Список всех финансовых записей.

get_total_by_category(category)

Возвращает сумму всех операций в указанной категории.

  • Параметры:
    • category: Категория для фильтрации (например, "Доход" или "Расход").
  • Возвращает: Сумму всех операций в указанной категории.

get_balance()

Возвращает текущий баланс (Доход - Расход).

  • Возвращает: Текущий баланс.

Пример использования


# Пример добавления финансовой записи
FinanceController.add_record(task_id=1, amount=1000, category='Доход')

# Получение всех записей
records = FinanceController.get_all_records()

# Получение суммы расходов
total_expense = FinanceController.get_total_by_category('Расход')

# Получение текущего баланса
current_balance = FinanceController.get_balance()
