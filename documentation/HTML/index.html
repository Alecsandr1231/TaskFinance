Документация TaskFinance

TaskFinance — это удобный и мощный планировщик задач с встроенной системой финансового учета и возможностью просмотра истории действий, разделенной по временным отрезкам.

  • Использование

Возможности

  • Создание, редактирование и удаление задач
  • Финансовые расчёты, связанные с задачами
  • История изменений с фильтрацией по временным периодам

Структура проекта

Проект разделен на три основных модуля:

  1. Task Manager (управление задачами) - Позволяет добавлять, редактировать, удалять задачи и отслеживать их статус.
  2. Finance Module (финансовые расчёты) - Рассчитывает бюджеты, доходы и расходы, связанные с задачами.
  3. History Tracker (история изменений) - Записывает и сортирует историю действий пользователей по временным отрезкам.

Установка

git clone https://github.com/Alecsandr1231/TaskFinance.git
cd taskfinance
pip install -r requirements.txt
