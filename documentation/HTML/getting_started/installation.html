Установка TaskFinance

Требования

  • Python 3.6 или выше
  • pip (пакетный менеджер для Python)
  • Git (для клонирования репозитория)

Шаги установки

  1. Клонирование репозитория

    Сначала клонируйте репозиторий TaskFinance на ваш локальный компьютер. Откройте терминал и выполните следующую команду:

    git clone https://github.com/your-repo/taskfinance.git

  2. Переход в директорию проекта

    Перейдите в директорию проекта:

    cd taskfinance

  3. Установка зависимостей

    Установите необходимые зависимости, используя pip:

    pip install -r requirements.txt

  4. Запуск приложения

    Теперь вы можете запустить приложение:

    python main.py

Проверка установки

После запуска приложения вы должны увидеть сообщение о том, что TaskFinance успешно запущен. Если возникли ошибки, проверьте, что все зависимости установлены правильно и что вы используете совместимую версию Python.

Дополнительная информация

Если у вас возникли вопросы или проблемы с установкой, пожалуйста, создайте issue в репозитории на GitHub или обратитесь к документации.
