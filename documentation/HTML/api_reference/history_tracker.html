Документация HistoryTracker

Обзор

Класс HistoryTracker отвечает за отслеживание и хранение истории действий пользователей в приложении TaskFinance. Он позволяет добавлять записи истории и получать их с возможностью фильтрации по дате.

Методы

add_record(task_id, action)

Добавляет новую запись истории действий.

  • Параметры:
    • task_id: Идентификатор задачи, с которой связана запись.
    • action: Описание действия, которое было выполнено.

get_records(start_date=None, end_date=None)

Возвращает список записей истории действий с возможностью фильтрации по дате.

  • Параметры:
    • start_date: (необязательный) Начальная дата для фильтрации.
    • end_date: (необязательный) Конечная дата для фильтрации.
  • Возвращает: Список записей истории действий, отфильтрованных по дате.

Пример использования


# Пример добавления записи истории
HistoryTracker.add_record(task_id=1, action='Задача создана')

# Получение всех записей истории
records = HistoryTracker.get_records()

# Получение записей за определенный период
filtered_records = HistoryTracker.get_records(start_date='2023-01-01 00:00:00', end_date='2023-12-31 23:59:59')
